# כרטיס היכולות של קרן — מה אפשר להגיד לסוכנת

מסמך זה הוא ההודעה שנשלחת לקרן (ולכל יוצרת תוכן חדשה) בוואטסאפ.
**כלל תחזוקה:** כל ניסוח כאן מאומת מול המזהים בקוד (confirmation / visibility / production-status). שינוי במזהה → חובה לעדכן את הכרטיס, ולהפך: אין להוסיף לכרטיס ניסוח שהקוד לא תופס.
עדכון אחרון: 12.7.2026.

## ההודעה (להעברה כמו שהיא)

היי קרן :)
רגע קטן של היכרות — הנה כל מה שאפשר להגיד לי, במילים שלך:

*רעיונות*
פשוט תכתבי לי רעיון, איך שבא לך: "יש לי רעיון לסרטון על..."
ואם זה טרנד שצריך לרוץ מהר: "טרנד: ..." — ואני מטפלת בו במסלול מהיר.

*אחרי שאני מציגה טיוטה*
אפשר לבקש כל שינוי: "תשני את הטון למצחיק", "עדיף פוסט ולא ריל", "תשני את השם ל..."
לשמור — פשוט "כן". להתחרט — "ביטול".

*כשמשהו מתקדם, ספרי לי*
"צילמתי את [שם]" / "ערכתי את [שם]" / "הקאבר של [שם] מוכן" / "העליתי את [שם]"
ככה אני יודעת בדיוק איפה כל דבר עומד.

*להזיז דברים קדימה*
"תוסיפי את [שם] להפקה" — וכשנצטרך תאריך, נשבץ יחד.
"בואי נתכנן את אוגוסט" — ונסדר את החודש ביחד.

*שאלות שאני יודעת לענות עליהן*
"מה דחוף?" / "מה הסטטוס של [שם]?" / "מה מתוכנן השבוע?" / "מה החורים בגאנט?"

*ארכיון*
"שימי את [שם] בצד" / "מה יש בארכיון?" / "תחזירי את [שם] מהארכיון"

*ואם לא הבנתי אותך* — נסחי שוב בפשטות, או כתבי "ביטול" ונתחיל נקי.

אני גם אכתוב לך לבד: בריף קצר בבוקר עם הפוקוס של היום, ותזכורת אחת אחר הצהריים. היעד שלנו: 2 רילסים בשבוע — ואני פה כדי שנעמוד בו ביחד.

## מיפוי ניסוח → מזהה (לתחזוקה בלבד)

| ניסוח בכרטיס | מזהה בקוד |
|---|---|
| "יש לי רעיון..." / טקסט חופשי | מסלול יצירת דראפט (ברירת מחדל בסוף השרשרת) |
| "טרנד: ..." | isTrendCommand (Fast Lane, קטגוריה טרנד) |
| "תשני את הטון/השם..." | isEditRequest → parseEditRequest / askClaudeForEdit |
| "כן" / "ביטול" | isConfirmationMessage / RESET_PATTERNS |
| "צילמתי/ערכתי/העליתי/הקאבר מוכן" | detectStatusUpdate (production-status) |
| "תוסיפי את X להפקה" | isApproveForProductionCommand |
| "בואי נתכנן את [חודש]" | monthly planning trigger |
| "מה דחוף/סטטוס/מתוכנן/חורים" | visibility intents + priority |
| "שימי בצד" / "מה יש בארכיון" / "תחזירי מהארכיון" | isArchiveCommand / isViewArchiveCommand / isRestoreCommand |
| "תשני את הדדליין של X ל-Y" | isDeadlineUpdate (לא בכרטיס של קרן בכוונה — פחות רלוונטי לה יומיומית) |
